실행 안내

지오로케이션은 보안 컨텍스트(https 또는 http://localhost)에서만 동작합니다. VS Code의 Live Server 확장이나 아래 명령으로 로컬 서버를 띄우세요:

python -m http.server 5500

그 다음 http://localhost:5500로 접속하세요.
